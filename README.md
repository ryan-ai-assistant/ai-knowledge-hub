# AI Knowledge Hub

Private, single source of truth for prompts, templates, frameworks, modules, and research.
Designed for **Mode C (Act & Assemble)** usage in ChatGPT.

## Structure
- `prompts/` – ready-to-use prompts (with placeholders)
- `prompts/_modules/` – tone, structure, style, disclaimers
- `prompts/_patterns/` – parameter conventions
- `templates/` – reusable document/email/Slack templates
- `frameworks/` – repeatable thinking structures (processes)
- `research/` – summaries & references
- `_inbox/` – drop new libraries here; ingestion normalizes & dedupes
- `catalog/` – auto-generated index
- `LOADER.md` – retrieval rules for assistants
